# Catalog API Contracts

## Media JSON rules

The `media` field accepts only plain JSON values:

- string, number, boolean, null
- array (recursive)
- object (recursive, plain objects only)

Prisma null sentinels are NOT accepted via the API:

- `Prisma.JsonNull`
- `Prisma.DbNull`
- `Prisma.AnyNull`

## Patch semantics

- Omit `media`: no change.
- Provide valid `media`: replace stored media with the provided value.

## Examples

Valid payloads:

```json
{ "media": null }
```

```json
{ "media": { "images": ["https://cdn.example.com/a.png"] } }
```

```json
{ "media": [true, 10, "ok"] }
```

Invalid inputs (server-side/SDK usage):

```ts
{ media: Prisma.JsonNull }
```

```ts
{ media: Prisma.DbNull }
```

```ts
{ media: new Date() }
```
